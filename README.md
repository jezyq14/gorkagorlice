# Górka Gorlice

Strona przeznaczona dla uczniów/rodziców/nauczycieli z [Zespołu Szkół nr 1 im. Ignacego Łukasiewicza w Gorlicach](https://lukasiewicz.gorlice.pl/).

Póki co znajduje się w stanie kwantowym, istnieje tylko w moim mózgu :)

## Planowane funkcjonalności

- Szczęśliwe numerki - scrapowanie ze strony szkoły i prowadzenie statystyk, teoretycznie możliwe by było zrobienie też ich przewidywanie na przyszłe dni.
- Plany lekcji - wyświetlanie ich w bardziej responsywnej formie niż [te na stronie szkoły](https://planlekcji.lukasiewicz.gorlice.pl/), plany lekcji nauczycieli.
- Składki - system dla skarbników klas ułatwiających zbieranie pieniędzy z powiadomieniami dla zalegających itp.
- Ogłoszenia - system ogłoszeń klasowych/samorządu szkolnego/globalnych.
- Zastępstwa - importowanie zastępstw z dziennika ([eduVULCAN](https://eduvulcan.pl/)) i ładne ich składanie w całość.

## Technologie

- Backend: Node.js, [Hono](https://hono.dev/), PostgreSQL ([Drizzle ORM](https://orm.drizzle.team/))
- Frontend: [SvelteKit](https://svelte.dev/), [shadcn-svelte](https://www.shadcn-svelte.com/)
